### Architecture

- **Extraction:** GPT-4o with v8.1 prompt
- **Axis Scoring:** 11 editorial axes (1-10) per product
- **Vibe Scoring:** Dot product of product axes × vibe weight vectors (12 vibes)
- **Confidence:** Per-attribute tiered thresholds (AUTO ≥90% / REVIEW 70-89% / MANUAL <70%)
- **Soft Demotions:** No hard blocks — all products can appear in any vibe
